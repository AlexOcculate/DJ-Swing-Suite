1. What is the DJ Project - Swing suite?

The DJ Project is a set of tools and libraries to enhance the user experience of
Java on the Desktop.

DJ Swing Suite provides some common components and utilities that are often
needed to create rich user interfaces.


2. How to use it?

Simply place the DJSwingSuite.jar library in your classpath.
Java 5.0 or later is required.


3. Any tutorial or demo?

The DJ Swing Suite Demo presents all the features of the Swing Suite library,
along with the corresponding code. Simply launch DJSwingSuite-Demo.jar.


4. What is the development status?

The library is tested on Windows and Linux, and logically works on all the
platforms where Java/Swing is supported.

For information about the current implementation status, visit the DJ Project's
website.


5. Sources?

The sources are part of the distribution.
There is of course some access to the CVS tree, from the Sourceforge website.


6. Troubleshooting?

The library is pretty simple. In case of a problem, contact me by e-mail or
through the SourceForge forums for the project.


7. How to contribute?

If you are interested in helping the project, simply send me an e-mail. Friendly
e-mails are always welcome too!
